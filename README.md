### Prerequisites

**Ncurses**

-Debian/Ubuntu Linux
  ```
  sudo apt-get install libncurses5-dev libncursesw5-dev
  ```
  
-CentOS/RHEL/Scientific Linux 6.x/7.x+ and Fedora Linux
 ```
  sudo yum install ncurses-devel
 ```
